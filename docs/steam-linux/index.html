jiezai-cbd 收录于 ACGN
 2022-04-24   约 270 字   预计阅读 1 分钟 

test

steam在linux上可以使用proton运行只支持windows的程序，然而我在使用时发现只有很少一部分游戏可以成功运行，还有很多游戏要么就是无法打开，要么就是打开后闪退，要么就是打开后报错，就连商店里标明的支持linux的游戏也有这种问题，这里记录解决的方式。 参考protondb.com

爱上火车:last run(maitetsu:last run!)

//支持平台 windows //打开后闪退,添加一下代码后成功运行，然而游戏只有6帧，不知道为什么 PROTON_USE_WINED3D11=1 %command%
